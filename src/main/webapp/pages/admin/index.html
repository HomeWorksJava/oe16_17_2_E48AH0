Kurzuskezelő felület

Jelenlegi funkcionalitás:

  • Kurzusok kezelése
    • Kurzusok kilistázása
    • Kurzus létrehozása
    • Kurzus módosítása
    • Kurzus törlése
    • Kurzusról kirúgás
  • Kurzustartalom kezelése
    • Kurzustartalmak kilistázása
    • Kurzustartalom létrehozása
    • Kurzustartalom módosítása
    • Kurzustartalom törlése
  • Kurzusra jelentkezés
    • Jelentkezés
    • Lejelentkezés
    • Jelentkezett kurzusok kilistázása
    • Jelentkezett kurzustartalmak megtekintése
  • Jelenlegi felhasználótípusok:
    • ADMIN
    • USER
  • Navigáció
    • Bejelentkezés
    • Kijelentkezés
    • Védett user/admin oldalak
    • Védett REST végpontok
