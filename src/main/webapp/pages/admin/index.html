Kurzuskezelő felület

Válassz a fenti menüpontok közül.

Jelenlegi funkcionalitás:

  • Kurzusok kezelése
    • Kurzusok kilistázása
    • Kurzus létrehozása
    • Kurzus módosítása
    • Kurzus törlése
    • Kurzusról kirúgás
  • Jelentkezés
    • Jelentkezés
    • Lejelentkezés
    • Jelentkezett kurzusok listázása
  • Jelenlegi felhasználótípusok:
    • ADMIN
    • USER
    • Teszt felhasználók (fh. : jelszó):
      • admin : admin
      • user1 : user
      • user2 : user
      • user3 : user
      • user4 : user
  • Navigáció
    • Bejelentkezés
    • Kijelentkezés
    • Védett user/admin oldalak
    • Védett REST végpontok
